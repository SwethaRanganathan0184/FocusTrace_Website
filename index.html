App Time Tracking
Every app session is tracked to the second. See exactly where your hours went, with category breakdowns and session counts.

Browser Tab Tracking
Click any browser in the list to drill down into the actual tabs you visited, grouped by category — Development, Entertainment, Social Media, and more.

AI Categorization
Tab titles are categorized automatically. Common sites use instant local matching. Unknown sites are sent to Groq AI and cached forever.

FocusTrace detects when you're in a meeting and tracks every time you switch away — so you can see how focused you actually were.

Simple setup

Up and running in under 2 minutes

1
Download
Download the app for Mac or Windows and install it like any other app.
2
Add API Key
Get a free Groq API key at console.groq.com and paste it in Settings on first launch.
3
Grant Permissions
Allow Accessibility and Screen Recording so FocusTrace can see which app is active.
4
Work Normally
FocusTrace runs silently in the background. Open the dashboard anytime to see your data.
